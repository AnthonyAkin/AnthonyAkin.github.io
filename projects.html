Orbit Simulation

with Euler Integration Methods

An Academic Assignment with the goal of assessing our fundamental understanding of C++ and OOP paradigms.


In particular, the project was focused on investigating the conditional stability, and computational speed, of various Euler Methods for computing Numerical Integration. The former section, completed on MATLAB, accessed a more Numerical approach analysising the cost and benefits of various methods. The latter section, focused on developing fast performing code on C++ making use of the 4th-Order Runge-Kutta Method in particular. This was the first major project I completed in C++, achieving a first overall in the module Computing and Numerical Methods 2.

Fluid Simulation

An Academic Assignment tasked at Parallelizing a provided Serial Code by our lead lecturer, in C++


This provided an introduction in the fundamentals of Computational Fluid Dynamics Solvers (CFDs), while developing an understanding of parallel paradigms. MPI and OpenMP were used and the fundamental parallelisation technique used was domain decomposition, hence requiring communication between processes but only for boundary data.

ISSH Antenna Sizing

A development on my Satellite Broadband Model Algorithm.


Reusing the code previously created in Project 1, I was tasked with simulating the internet capability of a proposed Space Station (the Imperial Space Station/Hotel a.k.a. ISSH). The Code needed to be expanded to iterate through time better, make use of a new Space Station Class, and allow basic antenna parameter sizing to speed up prelimery design.
